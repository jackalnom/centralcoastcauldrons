# Version 4 - Develop a Data-Driven Shop Strategy

## Visualize metrics

Visualize the following metrics:
1. A time series chart of sales per potion by hour. These should be the actual potions you've made available for sale so far. 
2. A table of all barrel types, when the barrel type is offered, what liquid type it contains, and the cost per ml.
3. Visualization(s) of what class, species, and level are purchasing what type of potion.
4. An additional visualization of your choosing that tells you something the above three visualizations don't.

Create a markdown named `metrics.md` that includes the SQL and the respective visualizations detailed above.

## Define your strategy
Create a markdown file called `strategy.md` and save it in your GitHub repository. In this document, outline your **hypothesis** on what you believe are the **most important changes** your shop needs to make to maximize profitability informed by the metrics you collected above. Be specific about why you believe these changes matter.
